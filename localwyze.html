LocalWyze

This was my third project at General Assembly. It was a group project that aimed to help people find items they would want to buy (often from far off venders), locally. So the group set out to create a place where business owners could post their items and customers could browse those items based on promity to them. My main role in this project was working with the google maps API and a subrole of styling.

Tech used:

HTML, CSS, Twitter BootStrap, JavaScript, Google maps API, Ruby on Rails, and PostgreSQL.
