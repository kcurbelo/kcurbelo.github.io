Hank & Volley

This was my second project at General Assembly. Hank & Volley, is a mock ecommerce website. I built it to see how a relational database functioned and the cool stuff that can be created with it. Understanding how one table in a database talked to another was a challenging concept for me to understand. Once I understood it everything after started falling into place. Planning the entire design of the project was key, so when it came time to code everyhting went relatively smooth.

Tech used:

HTML, CSS, Ruby on Rails, and PostgreSQL.
