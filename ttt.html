Tic-Tac-Toe

This was my first project at General Assembly. It was a 2-player Tic-Tac-Toe game. The most challeneging anf rewarding aspect of this project came from the game logic. A game that is straight forward is much different when actualy trying to put it in a language a computer understands. That is what made this project so fun. It was our first look at the possibilities of coding.

Tech used:

HTML, CSS, JavaScript, Angular, and Firebase.
