Barnacle

This was my final project at General Assembly. Barnacle pulls together relevant information about beaches to make searching for the right beach much easier. This is my so called "baby". I was super proud of this project because of all that was accomplished and has yet to be accomplished with this app. My goal for this project is to have it be on both app stores. To create this project I created my own API using rails and then used Ionic Framework to pull in that data to populate the app. It's a fun little app so be sure to look for it on the app store soon!

Tech used:

HTML, CSS, JavaScript, Ruby on Rails, Barnacle API, Angular, and Ionic Framework.
